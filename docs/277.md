# 删除行

> 原文:[https://www.learnsqlonline.org/en/Deleting_rows](https://www.learnsqlonline.org/en/Deleting_rows)

* * *

删除行与更新行非常相似，只是对行进行的那种类型的更新是删除。

下面是语法:

```sql
DELETE FROM table_name WHERE column1 = value1 AND column2 = value2 ... 
```

所有符合删除查询条件的行将被删除。
