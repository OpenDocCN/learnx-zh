算术运算符

可以使用算术表达式对变量进行简单的算术运算:$((expression))

```sh
A=3
B=$((100 * $A + 5)) # 305 
```

基本运算符有:

**a + b** 加法(a 加 b)

**a - b** 减法(a 减去 b)

**a * b** 乘法(a 乘以 b)

**a / b** 除法(整数)(a 除以 b)

**a % b** 取模(a 除以 b 的整数余数)

**a** ***** * **b** 取幂(a 到 b 的幂)
