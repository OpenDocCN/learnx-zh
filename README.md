## 下载

### Docker

```
docker pull apachecn0/learnx-zh
docker run -tid -p <port>:80 apachecn0/learnx-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g learnx-zh
learnx-zh <port>
# 访问 http://localhost:{port} 查看文档
```
